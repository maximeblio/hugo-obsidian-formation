title: "Aide-mémoire"
---

Les raccourcis principaux sont peu nombreux, mais permettent de gagner du temps et de fluidifier son utilisation. Il est également possible de paramétrer des raccourcis personnalisés.

## Raccourcis généraux

|Windows/Linux|Mac|Fonction|
| --- | --- | --- |
|`Ctrl + N`|`Cmd + N`|Créer une nouvelle note|
|`Ctrl + P`|`Cmd + P`|Ouvrir la palette de commande|
|`Ctrl + O`|`Cmd + O`|Recherche rapide par titre|
|`Ctrl+Shift+F`|`Cmd+Shift+F`|Recherche avancée dans tous les fichiers (y compris recherche *full text*)|
|`Ctrl + G`|`Cmd + G`|Ouvrir la vue graphique|
|`Ctrl + Alt + ←`|`Cmd + Alt + ←`|Naviguer en arrière
|`Ctrl + Alt + →`|`Cmd + Alt + →`|Naviguer vers l'avant|
|`Ctrl + F`|`Cmd + F`|Recherche dans le fichier courrant|
|`Ctrl + E`|`Cmd + E`|Basculer le mode édition / prévisualisation|
|`Ctrl + ,`|`Cmd + ,`|Ouvrir les paramètres généraux d'Obsidian|

## Raccourcis d'édition

|Windows/Linux|Mac|Fonction|
| --- | --- | --- |
|`Ctrl + B`|`Cmd + B`|Texte en **gras**|
|`Ctrl + I`|`Cmd + I`|Texte en *italique*|
|`Ctrl + K`|`Cmd + K`|Insérer un lien externe|
|`Ctrl + clic`|`Cmd + clic`|Ouvrir le lien d'une note dans un nouvel onglet|
|`Ctrl + ;`|`Cmd + ;`|Ouvrir les paramètres de la note|
|`Ctrl + H`|`Cmd + H`|Trouver/Remplacer un mot dans la note|

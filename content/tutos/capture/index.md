title: "Capturer des pages web"
date: 2025-11-07
---

Obsidian offre la possibilité de capturer des pages web pour les incorporer dans votre coffre. Le Markdown étant un dérivé du HTML, la note créée respectera les niveaux de titre, les styles d’écriture (**gras**, *italique* etc.) ainsi que l’incorporation des images, des liens hypertextes, etc.

### Installer le module sur votre navigateur

Comme pour Zotero, il convient d’ajouter dans votre navigateur le module, en passant par le site d’[Obsidian](https://obsidian.md/clipper).

Une fois ajouté, le module apparait dans la barre d’outil de votre navigateur.

### Enregistrer des pages

Si une page web vous intéresse, cliquez sur l’icone du module. Des champs de métadonnées seront préremplis, et la note s’ouvrira sur votre Obsidian.

<img src="/img/capture_1.gif">

Dans les paramètres du module, vous pouvez enregistrer des modèles (ajouter ou supprimer des métadonnées par exemple), gérer le comportement de la note générée etc.
